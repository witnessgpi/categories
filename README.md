# 站点使用指南
- 右上角的“事迹”“参考”按钮可以切换内容
- 左上角导航栏的`A`按钮可以调整文字
- 左上角导航栏的第一个按钮，可以折叠左边章节列表
- 左边章节列表的宽度可以调节
